# Changelog

## [0.1.0] - 2024-01-XX

### Added
- Реорганизация вопросов по категориям в отдельные файлы:
  - `src/data/questions/basics.ts` - Основы React (9 вопросов)
  - `src/data/questions/components.ts` - Компоненты (5 вопросов)
  - `src/data/questions/hooks.ts` - Hooks (10 вопросов)
  - `src/data/questions/state-management.ts` - State Management (5 вопросов)
  - `src/data/questions/routing.ts` - Роутинг (6 вопросов)
  - `src/data/questions/performance.ts` - Производительность (6 вопросов)

  - `src/data/questions/react18.ts` - React 18+ (7 вопросов)
  - `src/data/questions/questions.ts` - Объединяющий файл

### Fixed
- Исправлена логика подсчета вопросов в тестовых режимах - теперь режим всегда возвращает ровно указанное количество вопросов
- Устранено повторение вопросов в быстрых тестах - добавлен корректный алгоритм Fisher-Yates shuffle
- Режимы тестирования теперь дублируют вопросы если их недостаточно для достижения нужного количества

### Changed
- Упрощена структура проекта - вопросы теперь в отдельных файлах по категориям
- Обновлен README.md - убраны упоминания о 1000+ вопросах
- Обновлены описания в package.json и App.tsx
- Обновлен компонент TestModeSelector для правильного отображения режима "Полный тест"

### Technical
- Добавлена функция `shuffleArray` с использованием алгоритма Fisher-Yates для корректного перемешивания
- Улучшена функция `applyTestMode` для работы с недостаточным количеством вопросов
- Всего 54 уникальных вопроса покрывающих различные аспекты React
